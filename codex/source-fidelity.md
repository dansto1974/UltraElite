# Original Elite Fidelity Notes

Read this before changing behaviour intended to match original Elite.

## Rule

Use Mark Moxon's BBC Elite source as the first reference for source-faithful questions.

## Preserve Where Possible

- Galaxy/system identity.
- Economy, government, population, species, and goat-soup descriptions.
- Ship roles and broad combat/legal behaviour.
- Source-faithful mechanics when the game presents them as faithful.

## Ultra Departures

Ultra features may intentionally depart from the original when Dan has explicitly liked them. Call out the departure rather than pretending it is BBC-faithful.

## Known Source-Faithful Detail

Escape capsule use clears the player's legal status. The BBC `ESCAPE` routine explicitly sets `FIST` to `0` after emptying cargo and before consuming the pod.
