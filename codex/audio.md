# Audio Notes

Read this before sound, Sound Lab, procedural Web Audio, Classic/Ultra audio, lasers, explosions, engines, hyperspace, alarms, voice, or spatial audio work.

## Rules

- Classic mode should preserve simple BBC-style audio boundaries.
- Ultra mode can be cinematic, but sounds should be useful and not tiring.
- Use shared `eliteAudio` helpers and Sound Lab presets rather than isolated oscillator blocks.
- Persistent beds must be tested as loops, not only one-shot previews.
- Dan's ears win subjective sound calls.

## Workflow

- Make the smallest useful implementation change.
- Use Sound Lab when auditioning or tuning.
- Hand subjective listening back to Dan quickly rather than over-analysing generated audio.
- If an audio decision becomes durable, update this file or the `ultra-elite-audio-lab` skill.
